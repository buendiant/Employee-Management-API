# Employee-Management-API
This is an API created with Java, Spring Boot and SQL to manage employees at a imaginary enterprise

## What do you need to run it?

- have Eclipse Temurin JDK installed
- Have Spring Boot plugins in your IDE
- Use MySQL for the database

## How to run?

- Copy the whole project to your IDE
- Create a database in MySQL named empresa_db
- Replace "Your Password" in the "src/main/resources/application.properties" file with your actual root password
- Then, Run the EmployeeManagementApplication.java in your id
- Finally go to http://localhost:8080/api/employees, the page should run normally
- You should get the only employee that is in the database
